# CraftableSherds
Adds crafting recipes for pottery sherds


Each sherd crafting recipe uses a basic template as shown:

"CPC"

"GBG"

"CGC"

C: "minecraft:raw_copper_block"

P: pattern item

G: "minecraft:gilded_blackstone" -OR- "minecraft:raw_gold_block"

B: "minecraft:brick"
